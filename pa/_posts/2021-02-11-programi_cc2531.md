# ਪ੍ਰੋਗਰਾਮਿੰਗ ਕਰੋ

ਇਸਨੂੰ ਅਜ਼ਮਾਓ:
```bash
cd flash_cc2531
./cc_chipid
```
ਇਹ ਵਾਪਸ ਆਉਣਾ ਚਾਹੀਦਾ ਹੈ:
```
  ID = b524.
```

ਕੁੰਜੀ ਦੇ ਮੌਜੂਦਾ ਸਮਗਰੀ ਰੱਖੋ:
```bash
./cc_read save.hex
```

ਕੁੰਜੀ _CC2531_ਵਿੱਚ ਫਾਈਲ _CC2531ZNP-Prod.hex_ ਨੂੰ ਪ੍ਰੋਗਰਾਮ ਕਰੋ:
```bash
./cc_erase
./cc_write CC2531ZNP-Prod.hex
```
